# My Security

## Description
This is a simple security project using Spring Boot 3.0 Security.
It will include the following features:
- Authentication
- Authorization

## Installation
- Clone the project
- Run the following command to start the application
```bash
mvn spring-boot:run
```

## Usage
- Open the browser and go to the following URL
```bash
http://localhost:7000
```
